

l'application sert a vendre des items comme des pc portable par exemple

TP Anuglar version CLI 12.1 réaliser par Mlaouhia Bilel

Backend: json server 0.17.0 => to run server : json-server Nom_de_votre_base(format json) 
note: le fichiers backend doit être crée dans le dossier src et en dehors de fichier app

Frontend: Angular 12.1 + Bootstrap 5.0 + fontAwesome 4.7.0
note : to run server => ng serve -o

les modules utiliser dans cette application web:

- BrowserModule
- HttpClientModule
- AppRoutingModule
- FormsModule
- ReactiveFormsModule

Pipe crée: 
- prixDinar => affiche le montant original + TND => exp: 700 devenir 700 TND
note: pour creé une pipe taper ng g pipe Nom_Pipe

pagination: 
- avoir seulement le nombre des produits désiré a être afficher depuis la base (6 produits pour chaque page)

Gestion des erreures de saisie et messages de comminucation:
- grâce au modules de validation fournie par Angular (valid,invalid,dirty,pristine,touched,untouched) 

Icons :
- fournie par la bibliotéque font awsome
